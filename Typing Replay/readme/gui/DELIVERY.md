## 🚀 使用方式

### 方式 1: GUI 界面（推荐新手）

```bash
python gui.py
```

或使用启动器：

```bash
python start.py
# 选择选项 1: 启动 GUI 界面
```

### 方式 2: 命令行

```bash
# 运行示例
python examples.py 1

# 运行测试
python test_engine.py
```

### 方式 3: 编程集成

```python
from typing_replay import create_replay, type_text, pause

scheduler = create_replay([
    type_text("Hello!", wpm=60),
    pause(0.5)
])
scheduler.play()
```
